# Passport Module

身份驗證模組 - 提供登入、註冊等認證功能

## Structure

- `pages/` - Authentication page components
  - `login/` - Login page
  - `register/` - Register page
  - `lock/` - Lock screen page
  - `register-result/` - Registration result page
  - `callback.component.ts` - OAuth callback handler
- `routing/` - Route configuration

## Routes

- `/passport/login` - Login page
- `/passport/register` - Register page
- `/passport/lock` - Lock screen
- `/passport/register-result` - Registration result
- `/passport/callback` - OAuth callback
