Angular 核心套件: @angular/*
核心模組
@angular/core — 核心框架（Component / DI / Lifecycle 等）
@angular/common — 通用指令、管道、服務（NgIf, NgFor, DatePipe…）
@angular/compiler — 模板編譯器
@angular/forms — 表單支援（Template / Reactive Forms）
@angular/router — 路由系統與導航
@angular/platform-browser — 瀏覽器平台支援（DOM 操作）
@angular/platform-browser-dynamic — JIT 編譯支援
@angular/animations — 動畫支援
@angular/localize — 國際化（i18n）
CLI / 工具相關
@angular/cli — Angular CLI 工具
@angular/compiler-cli — 編譯支援（AOT / ngc）
@angular-devkit/build-angular — CLI build 架構
@angular-devkit/core — Devkit 核心工具
@angular-devkit/schematics — 代碼生成 / Schematics
可選官方套件
@angular/cdk — Component Dev Kit，低階 UI 工具（滾動、拖拽、鍵盤等）
@angular/elements — 將 Angular 元件封裝成 Web Components
@angular/platform-server — SSR / server-side rendering
@angular/service-worker — PWA / Service Worker
@angular/upgrade — 升級舊版 AngularJS 專案
@angular/language-service — Angular 語言服務支援（VSCode、IDE 自動提示）
